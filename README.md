## 📖 Overview

**StockGold** is a sophisticated desktop application tailored specifically for the management of jewelry stores, precious metals exchanges, and similar businesses. With an intuitive, modern user interface, it provides robust tools to handle end-to-end operations including inventory tracking, dynamic pricing, barcode generation, comprehensive sales & buys management, and secure financial logging.

## ✨ Key Features

- **📦 Inventory & Product Management**
  - Easily manage products, categories, types, and daily market prices.
  - Detailed cost, price tracking, and profit margin analysis.
  - Granular control over stock levels and scrap inventory.
  
- **🏷️ Intelligent Barcode System**
  - Built-in capabilities to generate, assign, and print physical barcodes.
  - Quick scanning integration for fast checkout and stock verification.
  - Dedicated pages for barcode drafting and customizable templates.

- **🛒 Advanced Point of Sale (POS)**
  - Seamless retail and wholesale transaction processing.
  - Manage *Sales*, *Buys*, and *Wholesaler* transactions seamlessly.
  - Real-time cart calculations with discounts and fluctuating market variables.

- **🏦 Safe & Cash Register Management**
  - **Instant Safe:** Real-time visibility into the store's current cash and metal balances.
  - Track day-to-day **Expenses**, **Profits**, and comprehensive cash flow.
  
- **🔐 User & System Security**
  - Role-based user authentication.
  - Local licensing checks and validation for secure software distribution.

## 🛠️ Tech Stack

StockGold relies on modern and reliable technologies to guarantee high performance and stability:

- **Framework:** [.NET 8](https://dotnet.microsoft.com/en-us/download/dotnet/8.0) 
- **UI Platform:** Windows Presentation Foundation (WPF) with an MVVM Architecture (using `MvvmLightLibs`)
- **Database:** Entity Framework Core (`EF Core 8.0`) connecting to a **MySQL** backend.
- **Barcodes:** `ZXing.Net` for reading and generating industrial-standard barcodes.
- **Miscellaneous Addons:** `X.PagedList` for efficient data navigation, `Newtonsoft.Json` for configuration, `Markdig` for text rendering.
